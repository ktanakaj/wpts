# リポジトリガイドライン

## プロジェクト概要
- このリポジトリは **Wikipediaの翻訳をサポートするためのWindowsのフリーソフト**です。

## ディレクトリ構成
- `HmLib` - 汎用的に使えそうな共通関数やクラスなど
- `Wptscs` - .NET Framework 4.7 Windows Formsアプリ
- `*Test` - 各プロジェクトのテストコード
- `Installer` - Wix 3.14のインストーラー用ファイル

## ビルド&テスト手順
- `nuget restore "./Wikipedia 翻訳支援ツール.sln"`: NuGetの取得
- `msbuild "./Wikipedia 翻訳支援ツール.sln"`: ビルド
- `pwsh -File .github/scripts/test.ps1`: ユニットテスト ※事前に要Debugビルド

## 実装ガイド
- 既存のソースコードの設計を参考にして、造りを合わせる。※ただし既存が露骨に間違っている場合は除く。
- 一般的なコーディングのベストプラクティス（命名, DRY, SOLID, KISS, 等）を意識する。
- Windows Forms, C#のベストプラクティスを意識する。
  - ただし以下はプロジェクト独自ルール優先。
    - 非同期メソッドに `XxxAsync()` のようなsuffixを付けない。
    - 長過ぎるコードは、`#region` で整理する。
- UIのテキストはローカライズ前提で全てリソース (resx) を使用する。
